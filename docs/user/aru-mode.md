# ARU Mode

!!! note "Coming Soon"
    This feature is under development and will be available in a future release.

ARU (Autonomous Recording Unit) Mode will transform your mobile device into a long-term periodic acoustic recording station (similar to Swift or AudioMoth hardware).

## Planned Features

- **Recording Schedules**: Schedule active recording windows (e.g., "10 minutes every hour" or "from sunrise to sunset").
- **GPS & Weather Capture**: Auto-geotag deployments and capture local weather metadata at initialization.
- **Power Optimization**: Instructions and checks to optimize battery life (such as manually enabling Airplane Mode) and keeping the audio pipeline robust in the background.
- **Storage Metrics**: Calculate and check estimated storage needs and available system memory before deploying.
- **Session Consolidation**: Group recorded segments into either separate sessions or one combined parent session.

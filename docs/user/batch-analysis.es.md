# Análisis por lotes

!!! note "Próximamente"
    Esta función está en desarrollo y estará disponible en una versión futura.

El análisis por lotes permitirá procesar y analizar varios archivos de audio en lote (por ejemplo, desde una tarjeta SD conectada a través de un lector USB) utilizando parámetros de análisis personalizados.

## Funciones planificadas

- **Acceso a almacenamiento externo**: Escanee y lea carpetas directamente desde tarjetas SD externas o unidades USB.
- **Procesamiento por lotes**: Analice directorios completos de archivos de audio en una sola ejecución por lotes.
- **Exportación flexible**: Guarde los resultados directamente en el almacenamiento externo en formatos Raven Selection Table, CSV, TSV o JSON.
- **Submuestreo de sesiones**: Opción para importar resultados en la base de datos de la aplicación utilizando filtros de submuestreo inteligentes para evitar la saturación de la base de datos.

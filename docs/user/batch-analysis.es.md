# Análisis por lotes

!!! note "Próximamente"
    Esta función está en desarrollo y estará disponible en una versión futura.

El Análisis por lotes permitirá procesar y analizar múltiples archivos de audio a la vez (por ejemplo, desde una tarjeta SD conectada a través de un lector USB) usando parámetros de análisis personalizados.

## Funciones planificadas

- **Acceso a almacenamiento externo**: escanea y lee carpetas directamente desde tarjetas SD externas o unidades USB.
- **Procesamiento por lotes**: analiza directorios completos de archivos de audio en una sola ejecución por lotes.
- **Exportación flexible**: guarda los resultados directamente en el almacenamiento externo en formatos Raven Selection Table, CSV, TSV o JSON.
- **Submuestreo de sesiones**: opción para importar los resultados a la base de datos de la app usando filtros de submuestreo inteligentes para evitar la saturación de la base de datos.

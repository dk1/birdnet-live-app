# Analyse par lots

!!! note "Prochainement"
    Cette fonctionnalité est en cours de développement et sera disponible dans une future version.

L'Analyse par lots vous permettra de traiter et d'analyser plusieurs fichiers audio à la fois (par exemple, à partir d'une carte SD connectée via un lecteur USB) avec des paramètres d'analyse personnalisés.

## Fonctionnalités planifiées

- **Accès au stockage externe**: Scannez et lisez des dossiers directement depuis des cartes SD externes ou des clés USB.
- **Traitement par lots**: Analysez des répertoires entiers de fichiers audio en une seule fois.
- **Exportation flexible**: Enregistrez les résultats directement sur un stockage externe aux formats Raven Selection Table, CSV, TSV ou JSON.
- **Sous-échantillonnage de session**: Option pour importer les résultats dans la base de données de l'application en utilisant des filtres de sous-échantillonnage intelligents pour éviter l'encombrement de la base de données.

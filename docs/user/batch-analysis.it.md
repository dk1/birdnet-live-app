# Analisi in batch

!!! note "Prossimamente"
    Questa funzione è in fase di sviluppo e sarà disponibile in una versione futura.

L'analisi in batch consentirà agli utenti di elaborare e analizzare più file audio in batch (ad esempio, da una scheda SD collegata tramite un lettore USB) utilizzando parametri di analisi personalizzati.

## Funzioni pianificate

- **Accesso alla memoria esterna**: Scansiona e leggi cartelle direttamente da schede SD esterne o chiavette USB.
- **Elaborazione in batch**: Analizza intere directory di file audio in un'unica esecuzione in batch.
- **Esportazione flessibile**: Salva i risultati direttamente sulla memoria esterna nei formati Raven Selection Table, CSV, TSV o JSON.
- **Sottocampionamento della sessione**: Opzione per importare i risultati nel database dell'app utilizzando filtri di sottocampionamento intelligenti per evitare il sovraccarico del database.

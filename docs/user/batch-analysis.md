# Batch Analysis

!!! note "Coming Soon"
    This feature is under development and will be available in a future release.

Batch Analysis will allow users to process and analyze multiple audio files in bulk (e.g., from an SD card connected via a USB reader) using custom analysis parameters.

## Planned Features

- **External Storage Access**: Scan and read folders directly from external SD cards or USB drives.
- **Bulk Processing**: Analyze entire directories of audio files in a single batch run.
- **Flexible Export**: Save results directly to external storage in Raven Selection Table, CSV, TSV, or JSON formats.
- **Session Subsampling**: Option to import results into the app database using smart subsampling filters to prevent database bloating.

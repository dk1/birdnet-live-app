# Explorar

Explore muestra las especies previstas para la ubicación y temporada actuales utilizando el modelo geográfico BirdNET.

## Cómo abrirlo

Abra **Explorar** desde el pie de página de Inicio usando el botón :material-magnify:.

## Barra de aplicaciones y encabezado

### barra de aplicaciones

- :material-refresh: — actualiza la ubicación y reconstruye la lista de especies previstas

### Encabezado de ubicación

El encabezado muestra:

- nombre del lugar actual con codificación geográfica inversa cuando esté disponible
- coordenadas debajo del nombre del lugar
- :material-help-circle-outline: — abre la hoja de ayuda de Explorar

## Lista de especies

Cada ficha de especie puede incluir:

- imagen de especies agrupadas
- nombre común
- nombre científico opcional
- insignia de puntuación de modelo geográfico

Toque una tarjeta para abrir la superposición de detalles de la especie.

## Superposición de detalles de especies

La superposición puede mostrar:

- imagen más grande
- crédito de imagen
- nombres comunes y científicos
- texto de descripción incluido cuando esté disponible
- gráfico semanal de frecuencia esperada
- enlaces externos como eBird, iNaturalist o Wikipedia cuando estén disponibles para esa especie

## Para qué sirve Explorar

Explorar es una vista de referencia con reconocimiento de ubicación dentro de la aplicación. Le ayuda a comparar el contexto de ubicación actual de la aplicación con las especies que espera encontrar.

**No** cambia los datos de la sesión guardada por sí solo. El filtrado de detección se controla por separado a través de [Configuración] (settings.md).

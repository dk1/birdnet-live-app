# Análisis de archivos

File Analysis procesa una grabación existente a través del mismo canal BirdNET utilizado por los flujos de trabajo en vivo.

## Cómo abrirlo

Desde Inicio, toque la tarjeta **Análisis de archivos** con el ícono :material-file-music:.

## Barra de aplicaciones

- :material-tune: — abre la configuración de Análisis de archivos
- :material-close: — cancelar una ejecución de análisis activa

## Entradas admitidas

El selector de archivos actual acepta:

- WAV / ONDA
-FLAC
- MP3
-OGG/OGA/Opus
-M4A/AAC/MP4
- WMA / AMR

## Asistente de cuatro pasos

### 1. Seleccionar archivo

Elija un archivo y revise su tarjeta de metadatos:

- nombre del archivo
- formato
- duración
- tamaño del archivo
- frecuencia de muestreo

### 2. Ubicación y fecha

Puede:

- utilizar GPS actual
- introducir coordenadas manualmente
- saltar ubicación
- elige un punto en el mapa
- establecer una fecha de grabación opcional

### 3. Parámetros

El asistente expone:

- duración de la ventana
- superposición
- sensibilidad
- umbral de confianza
- modo de filtro de especies

### 4. Analizar

La pantalla de progreso muestra:

- ventanas procesadas
- detecciones encontradas
- especies encontradas
- botón cancelar

## Resultado

Cuando finaliza el análisis, BirdNET Live convierte el resultado en una sesión guardada y abre [Revisión de sesión] (session-review.md).

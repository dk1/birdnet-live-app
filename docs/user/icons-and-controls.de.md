# Symbole und Steuerelemente

Auf dieser Seite werden die wiederkehrenden Steuerelemente und Symbole erläutert, die in BirdNET Live verwendet werden. Die folgenden Beschriftungen stimmen mit Steuerelementen überein, die bereits in der App vorhanden sind.

## Gemeinsame Navigationssteuerelemente

| Kontrolle | Wo Sie es sehen | Was es tut |
|---|---|---|
| :material-tune: **Einstellungen** | Start-Fußzeile, Live, Punktezählung, Umfrage, Dateianalyse, Sitzungsrückblick | Öffnet Einstellungen. In den Modusbildschirmen werden die für den jeweiligen Arbeitsablauf relevantesten Einstellungen geöffnet. |
| :material-magnify: **Erkunden** | Start-Fußzeile | Öffnet „Erkunden“. |
| :material-music-box-multiple-outline: **Bibliothek** | Start-Fußzeile | Öffnet die Sitzungsbibliothek. |
| :material-help-circle-outline: **Hilfe** | Start-Fußzeile, Kopfzeile „Erkunden“, Umfrage-Dashboard, Symbolleiste „Sitzungsüberprüfung“ | Öffnet die Hilfe oder ein bildschirmspezifisches Hilfeblatt. |
| :material-information-outline: **Info / Über** | Start-Fußzeile, Infoleisten, Hilfeblätter | Zeigt allgemeine Informationen oder einen zusammenfassenden Kontext an. |
| :material-arrow-left: **Zurück** | Live-Modus | Kehrt zum vorherigen Bildschirm zurück. |
| :material-open-in-new: **Extern öffnen** | Über Bildschirm, Dokumentationslinks | Öffnet eine externe Seite, beispielsweise das Online-Benutzerhandbuch. |
| :material-volunteer-activism: **Spenden** | Über-Bildschirm | Öffnet die BirdNET-Spendenseite. |

## Start-, Stopp- und Sitzungssteuerung

| Kontrolle | Bedeutung |
|---|---|
| :material-microphone: **Mic** | Beginnen Sie mit dem Live-Hören. |
| :material-stop: **Stop** | Stoppen Sie eine aktive Aufzeichnung, Punktzählung oder Umfrage. |
| :material-play: **Play** | Starten Sie einen konfigurierten Setup-Ablauf oder setzen Sie ihn aus einem angehaltenen Bereitschaftszustand fort. |
| :material-close: **Schließen / Abbrechen** | Brechen Sie eine aktive Dateianalyse ab. |
| :material-timer: **Timer** | Dauer oder verbleibende Zeit. |
| :material-alert-circle-outline: **Fehler** | Modell- oder Verarbeitungsfehler. |

## Orts- und Zeitkontrollen

| Kontrolle | Bedeutung |
|---|---|
| :material-crosshairs-gps: **Aktueller Standort** | Verwenden Sie die aktuelle GPS-Position des Geräts. |
| :material-map-marker-plus: **Manuelle Koordinaten** | Geben Sie die Koordinaten manuell ein. |
| :material-map-marker-off: **Kein Standort** | Standort überspringen oder anzeigen, dass der Standort nicht verfügbar ist. |
| :material-map-marker: **Hat Standort** | Bestätigen Sie einen Standort, zeigen Sie Koordinaten an oder kennzeichnen Sie eine kartierte Sitzung. |
| :material-refresh: **Aktualisieren** | Lesen Sie den aktuellen Standort erneut oder aktualisieren Sie eine Vorhersageliste. |
| :material-map: **Kartenauswahl** | Wählen Sie Koordinaten aus der Kartenauswahl aus. |
| :material-kalender: **Datum** | Datum festlegen oder anzeigen. |
| :material-close: **Klar** | Ein ausgewähltes Datum entfernen. |

## Erkunden und Artensymbole

| Kontrolle | Bedeutung |
|---|---|
| Miniaturansicht der Art | Gebündeltes Bild für die Art, sofern verfügbar. |
| Vertrauens- oder Geomodell-Prozentabzeichen | Eine kurze numerische Zusammenfassung der Modellausgabe. Höhere Zahlen weisen auf eine stärkere Unterstützung im Kontext dieses Bildschirms hin. |
| Monatsetiketten („Jan“, „Apr“, „Jul“, „Okt“, „Dez“) | Referenzpunkte auf dem wöchentlichen Diagramm der erwarteten Häufigkeit im Arten-Overlay. |

## Sitzungsüberprüfungssymbolleiste

Diese Steuerelemente werden auf dem Bildschirm „Sitzungsüberprüfung“ verwendet.

| Kontrolle | Bedeutung |
|---|---|
| :material-plus-circle-outline: **Hinzufügen** | Fügen Sie Inhalte hinzu, z. B. eine Art oder eine Anmerkung. |
| :material-undo-variant: **Rückgängig machen** / :material-redo-variant: **Wiederherstellen** | Gehen Sie durch die Überprüfungsänderungen vor oder zurück. |
| :material-content-cut: **Trim** | Wechseln Sie in den Trimmmodus oder zeigen Sie an, dass der Trimmmodus aktiv ist. |
| :material-content-save: **Speichern** | Bewertungsänderungen speichern. |
| :material-share-variant: **Teilen** | Exportieren oder teilen Sie die Sitzung. |
| :material-delete-outline: **Löschen** | Verwerfen Sie die Sitzung. |
| :material-play: **Weiter** | Setzen Sie eine noch nicht abgeschlossene Umfrage aus der Sitzungsüberprüfung fort, wenn diese Aktion verfügbar ist. |

## Bildschirmspezifische Statusleisten

### Live-Modus

Die Live-Infoleiste verwendet :material-information-outline: gefolgt von kompakten Beschriftungen wie:

- „Jetzt“ – Detektionen, die derzeit in der Live-Liste sichtbar sind
- „spp“ – einzigartige Artenzahl
- „det“ – Gesamtzahl der Detektionen
- Dauer und geschätzte Aufnahmegröße, wenn die Aufnahme aktiv ist

### Punkteanzahl

Die Punktezähler-Timerleiste kombiniert :material-stop: **Stop**, :material-timer: **Timer** und einen Fortschrittsbalken, um die verbleibende zeitgesteuerte Sitzung anzuzeigen.

### Umfrage

Das Umfrage-Dashboard verwendet:

- :material-map-outline: **Karte** – Registerkarte „Live-Karte“.
- :material-equalizer: **Spektrogramm** – Registerkarte „Spektrogramm“.
- :material-chart-bar: **Zusammenfassung** – Registerkarte „Zusammenfassung“.
- :material-chart-bar: Statistikbeschriftungen in der Übersichtsansicht der Umfrage

## Im Zweifelsfall

Wenn Sie nicht sicher sind, was ein Steuerelement bewirkt, öffnen Sie das nächstgelegene Hilfeblatt in der App oder sehen Sie sich die Workflow-Seite für diesen Bildschirm in diesem Benutzerhandbuch an.

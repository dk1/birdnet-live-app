# User Guide

This guide explains BirdNET Live as a field workflow tool: what each screen does, what the recurring icons mean, and how the settings shape the app's behavior.

It is written for ecologists and bird survey practitioners who already know the field methods and simply want a clear map of the app itself.

### [Icons & Controls](icons-and-controls.md)

### [Settings](settings.md)

Every setting currently exposed in the app, grouped the same way as the in-app Settings screen.

## Reading Order

1. Read [Getting Started](getting-started.md).
2. Skim [Icons & Controls](icons-and-controls.md) so the recurring symbols make sense.
3. Open the guide for the mode you use most often.
4. Use [Settings](settings.md) when you need to tune the app for a different workflow.

# Mode de comptage de points

Le mode Point Count est le flux de travail stationnaire chronométré dans BirdNET Live.

## Comment l'ouvrir

Depuis Accueil, appuyez sur la carte **Point Count Mode** avec l'icône :material-map-marker:.

## Flux de configuration

La configuration du nombre de points utilise trois étapes.

### 1. Durée et lieu

Choisir:

- un des jetons de durée disponibles
- GPS actuel avec :material-crosshairs-gps :
- coordonnées manuelles avec :material-map-marker-plus :
- aucun emplacement avec :material-map-marker-off :
- sélecteur de carte avec :material-map :

### 2. Conseils sur le terrain

Cet écran fournit une courte liste de contrôle dans l'application avant de commencer.

### 3. Prêt

L'écran prêt résume la durée sélectionnée et vous permet de commencer avec :material-play:.

## Écran de décompte de points en direct

L'écran de comptage en direct se concentre sur un tableau de bord chronométré.

### Barre supérieure

- :material-stop: — termine le décompte de points plus tôt
- :material-timer : - afficher le temps restant
- :material-tune : — ouvre les paramètres de comptage de points

### Principaux indicateurs

- barre de progression du compte à rebours
- barre d'informations compacte avec les détections actuelles, le nombre d'espèces uniques et le nombre total de détections
- vue spectrogramme
- liste de détection

## Après le décompte

Lorsque le décompte des points se termine, BirdNET Live enregistre la session et ouvre [Session Review] (session-review.md).

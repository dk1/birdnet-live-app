# Bibliothèque de sessions

La bibliothèque de sessions est l'archive des sessions enregistrées et des fichiers traités.

## Comment l'ouvrir

Utilisez le bouton :material-music-box-multiple-outline: dans le pied de page Accueil.

## Ce que montre la bibliothèque

Chaque entrée de session résume un ensemble de résultats enregistré, y compris son type, sa date, sa durée, le nombre d'espèces et le nombre de détections.

Les types de sessions utilisent les mêmes icônes que l'écran d'accueil :

- :matériel-microphone : — Session live
- :material-file-music: — Session d'analyse de fichiers
- :material-map-marker : – Séance de comptage de points
- :material-routes: — Session d'enquête

## Contrôles de la barre d'application

- :material-magnify : — recherche par date, type de session, nom de lieu, coordonnées, nom commun ou nom scientifique
- Menu du mode d'affichage : basculez entre **Détaillé**, **Compact** et **Par espèces**
- :material-swap-vertical: — change l'ordre de tri

## Modes d'affichage

### Détaillé

Affiche les cartes de session complètes avec plus de métadonnées.

### Compact

Affiche des lignes plus serrées pour une navigation plus rapide.

### Par espèce

Regroupe les sessions par espèce et s'étend aux sessions qui contiennent cette espèce.

## Que se passe-t-il ensuite

Appuyez sur n'importe quelle session pour ouvrir [Révision de session] (session-review.md).

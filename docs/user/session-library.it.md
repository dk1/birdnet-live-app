# Libreria di sessione

La Session Library è l'archivio delle sessioni salvate e dei file elaborati.

## Come aprirlo

Utilizza il pulsante :material-music-box-multiple-outline: nel piè di pagina Home.

## Cosa mostra la Biblioteca

Ciascuna voce di sessione riepiloga un set di risultati salvato, inclusi tipo, data, durata, conteggio delle specie e conteggio dei rilevamenti.

I tipi di sessione utilizzano le stesse icone della schermata Home:

- :material-microphone: — Sessione dal vivo
- :material-file-music: — Sessione di analisi dei file
- :material-map-marker: - Sessione di conteggio punti
- :material-route: — Sessione di indagine

## Controlli della barra dell'app

- :material-magnify: - ricerca per data, tipo di sessione, nome del luogo, coordinate, nome comune o nome scientifico
- menu della modalità di visualizzazione: alterna tra **Dettagliato**, **Compatto** e **Per specie**
- :material-swap-vertical: — cambia l'ordinamento

## Modalità di visualizzazione

### Dettagliato

Mostra le schede della sessione completa con più metadati.

### Compatto

Mostra righe più strette per una navigazione più rapida.

### Per specie

Raggruppa le sessioni per specie ed espande alle sessioni che contengono quella specie.

## Cosa succede dopo

Tocca qualsiasi sessione per aprire [Revisione sessione](session-review.md).

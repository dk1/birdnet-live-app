# Biblioteca de sessões

Biblioteca de Sessões é o arquivo de sessões salvas e arquivos processados.

## Como abri-lo

Use o botão :material-music-box-multiple-outline: no rodapé inicial.

## O que a biblioteca mostra

Cada entrada de sessão resume um conjunto de resultados salvo, incluindo tipo, data, duração, contagem de espécies e contagem de detecção.

Os tipos de sessão usam os mesmos ícones da tela inicial:

- :material-microphone: — Sessão ao vivo
- :material-file-music: — Sessão de análise de arquivo
- :material-map-marker: — Sessão de contagem de pontos
- :material-routes: — Sessão de pesquisa

## Controles da barra de aplicativos

- :material-magnify: - pesquisa por data, tipo de sessão, nome do local, coordenadas, nome comum ou nome científico
- menu do modo de visualização - alterne entre **Detalhado**, **Compacto** e **Por espécie**
- :material-swap-vertical: — altera a ordem de classificação

## Modos de visualização

### Detalhado

Mostra cartões de sessão completos com mais metadados.

### Compacto

Mostra linhas mais compactas para uma navegação mais rápida.

### Por espécie

Agrupa sessões por espécie e expande para as sessões que contêm essa espécie.

## O que acontece a seguir

Toque em qualquer sessão para abrir [Revisão da sessão](session-review.md).
